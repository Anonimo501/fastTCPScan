Parrot OS 4.11

## Install

git clone https://github.com/Anonimo501/fastTCPScan.git

cd fastTCPScan/

go build -ldflags "-s -w" fastTCPScan.go

upx brute fastTCPScan

## Uso

./fastTCPScan -h

./fastTCPScan -host 192.168.43.1 -threads 20
